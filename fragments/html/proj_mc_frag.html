Audio Visualizer (Java/Processing, Personal Project)

This is captured video from an audio visualizer I've been working on using Java, Processing, and the Minim library for audio I/O and analytics. Each individual animated circle is set to respond to the amplitude of a specific frequency range, with larger amplitudes propelling them outwards before gravity pulls them back towards the center. The audio source used for this video is an MP3 file (one of my favorite songs of all time, "Run Away With Me" by Carly Rae Jepsen) but the visualizer can respond in real time to audio sampled from a microphone input. I also added motion blur and bloom effects that I implemented myself to give it a higher-quality feel. Some additional videos I've nade with this project are below.
