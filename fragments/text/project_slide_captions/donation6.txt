By tapping the profile image of the charity from one of the previous posts, the user can see more in-depth information about specific charities, which can also be browsed outside the feed via the "Discover Charities and Foundations" section in the home screen. From the "Profile" tab on the charity page, the user can see the charity's self-submitted information and description and feed posts, as well as decide to donate to the charity now or automatically via future purposes.
